Wear Os

手表直接连接传感器：此选项使手表与传感器直接连接。仅在手表和手机同步时可更改此设置。使用“镜像 -> 同步并重新初始化”在手机和手表上同步手表和手机。

初始化手表应用程序：在手表上安装Juggluco(Wear Os版)并运行后，需要按下此按钮。

恢复默认：将连接设置为其默认值。它包括左侧菜单 -> 镜像（手表上）和左中部菜单 -> 镜像（手机上）中更改的设置。它包括诸如“仅激活”和“扫描”之类的设置。如果因淋浴或睡眠时不慎更改了设置，则可以按下此按钮。如果通过重新安装手表版本或清除数据重置，则还需要在手机上移除镜像连接。

特殊版本的Juggluco可以直接在Wear Os手表上运行。它需要一台带有NFC的Android智能手机来扫描传感器，并通过TCP连接将数据发送到Wear OS手表。之后你可以选择：

  1. 默认传感器与手机连接，并且通过蓝牙将接收到的血糖值通过TCP发送到Wear OS手表；

  2. 或者也可以选择将传感器直接与Wear OS手表连接。为此，需要具有良好蓝牙连接性的Wear OS手表。

一些图片在此处显示：https://www.juggluco.nl/JugglucoWearOS

要使其正常工作，请执行以下操作：

  • 确保手表和手机的蓝牙和WIFI正常工作；

  • 确保手表具有足够的可用存储空间和内存。如果设备在关键时刻存储空间不足，可能需要重新安装Juggluco；

  • 让Juggluco手机版本与传感器配对。您需要使用“通过蓝牙扫描传感器”多次扫描传感器。无需通过蓝牙在手机上接收血糖值；

  • 在Wear Os手表上安装Juggluco Wear OS版本并启动它；

  • 打开手机上的左侧菜单 -> 手表 -> Wear Os配置。您的手表应显示在列表中。按下“初始化手表应用程序”；

  • 如果一切顺利，手机和手表会成功连接，并且手机上的所有数据都会发送到手表。这可能需要很长时间，如果您使用Juggluco已经很长时间了。您可以通过查看镜像对话框（手机：左中部菜单 -> 镜像，手表：左侧菜单 -> 镜像）。应该显示与手表下拉列表中显示的相同ID的连接。在手表上也应包含手机的IP地址。如果这两者不在或者手表侧不包含IP，则应该再次按下“初始化手表应用程序”。如果仍然不行，请检查蓝牙连接或与手表连接的手机app。（当手表已经接收到数据时，按下“初始化手表应用程序”将重发所有数据。

  • 如果一切顺利，手机版本和手表版本的Juggluco显示相同的数据。手机接收的每个血糖值都会通过TCP发送到手表。一些设置，例如单位，血糖警报级别和提醒也会传输到手表。如果想要修改请在手表本身上操作。

  • 如果您有蓝牙连接良好的Wear Os手表，并且希望在不携带手机的情况下在手表上接收血糖值，可以直接将Juggluco手表版本与传感器连接。在将所有值发送到手表后，您可以勾选“手表直接连接传感器”。通过此操作，在手机上关闭“通过蓝牙扫描传感器”，并在手表上打开它。观看的流式值现在会从手表发送到手机，您可以在镜像对话框中检查。在手表和手机不同步时无法更改“手表直接连接传感器”设置。

手表-手机连接

手表-手机连接是在运行Juggluco的手机和运行Juggluco服务器的计算机之间进行的镜像连接的改编版本。当以上述方式自动生成手机和手表之间的连接时，它具有以下附加功能：

  • 自动确定连接的另一端的IP。通常情况下，您无需直接处理镜像连接。相反，您可以使用左侧菜单→手表→Wear OS配置来更改设置。唯一的例外是当您希望在手表上输入数字时。然后，您必须手动更改镜像设置，如下所述：在手表上输入。

  • 如果手机和手表之间的WIFI连接不起作用，Juggluco将切换到使用蓝牙上的TCP。仅当向手表发送大量数据时，您可以关闭手表上的WIFI。

当电话和手表上的Juggluco之间没有交换数据时，您可以查看以下内容：

  • 手表和手机上应该打开蓝牙（可以关闭WIFI）；

  • 手表的伴侣应用程序应显示手表已连接到手机；

  • 在手表上的左侧菜单→镜像和手机上的左中部菜单→镜像应自动创建具有手表标识符的连接条目（与手机上的左侧菜单→手表→“Wear OS配置”中显示的标识符相同）。

  • 在手表上的镜像菜单中，重新初始化和/或同步可能有所帮助；

  • 有时您需要在手机上执行相同的操作。然后，您可能需要再次在手表上执行相同操作；

  • 在手机和手表上轮流关闭和打开蓝牙可能会有所帮助。此后，您还需要在手表上按下重新初始化和同步；

  • 使用WIFI或移动数据执行相同操作有时是必要的；

  • 有时候，需要重新启动手机和/或手表，以使蓝牙重新正常工作；

  • 可能以某种方式更改了镜像连接中的设置，导致其值错误。可以通过在手机上按下左侧菜单→手表→“Wear OS配置”→默认值来将其重置为原始值。这也会将传感器连接到手机而不是手表。将重写仅在手表上存在的数据；

  • 如果所有其他方法都失败，则可以重新安装手表版本并重新开始。在这种情况下，您需要在左中部菜单→镜像下删除现有的手表连接。

  • 如果以某种方式手机上的数据丢失，但在手表上存在，则情况就会更加困难。您必须首先创建一个功能正常的镜像连接，以将数据接收到手机上。

在手表上输入

如果您在手表上输入胰岛素、碳水化合物和活动量，则将无法在手机上进行输入。另外，无法扫描胰岛素笔。

当前没有直接的按钮可用于此目的。您必须在连接手表和手机之间的镜像连接的两侧进行更改镜像设置。首先确保手机和手表同步，然后执行以下操作：

手机：

进入左中部菜单→镜像。按下手表连接，然后点击修改。取消“标签”的选择，并确保选中接收自。然后按保存然后按关闭。设置镜像发送“标签”。

手表：

进入左侧菜单→镜像。按下手表连接，然后点击修改。设置“标签”。将“现在”设置为已在手机上存在所有数据的时间点。然后按保存然后按关闭。取消选择镜像发送“标签”。

表盘

Wear OS的Juggluco包括一个表盘，可以显示当前的血糖水平、时间和最多四个复杂功能。要使用它，您必须选择此表盘，例如在手表伴侣应用程序中选择"已下载"。当屏幕唤醒时，通常会显示最后的血糖值。当显示处于环境（始终显示）模式时，只有在分钟指示器增加或按下按钮或抬起手臂时，血糖值才会更新。因此，显示的值可能会延迟高达一分钟。

还有一种复杂功能使用WearOS版本的Glucodata广播显示血糖值，可以在接受此复杂功能的每个表盘上使用：https://github.com/pachi81/GlucoDataHandler/releases

第三个选项是在 Juggluco 手表的设置中打开 Floating Glucose。当 Juggluco 有适当的权限时，一个血糖值会显示在其他应用程序上方。这样，您在使用任意的表盘或监测您活动的应用程序时，可以看到您当前的血糖值。有关更多信息，请参见 Juggluco 手机版本中的左侧菜单 -> 设置 -> Floating glucose -> 帮助。
